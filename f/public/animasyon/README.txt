Ana sayfa arka plan animasyonu dosyaları

Bu klasöre koyabileceğiniz dosyalar:
- Lottie: .json (örn. lottie-bg.json)
- Video: .mp4, .webm (örn. bg-loop.mp4)
- GIF: .gif

Kodda referans:
- public/animasyon/dosya.json  →  /animasyon/dosya.json
- public/animasyon/bg.mp4       →  /animasyon/bg.mp4

Örnek (video bg): <video src="/animasyon/bg.mp4" ... />
Örnek (Lottie): lottie.loadAnimation({ path: "/animasyon/lottie-bg.json", ... })
